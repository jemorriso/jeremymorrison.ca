- Python client that accesses Sportsbook Review's GraphQL endpoint and returns cleaned and formatted data
- Postman and GraphQL introspection used to discover the structure of the schema
- Currently the only free and open source library that can get odds from over 30 sportsbooks
- Automated test suite using PyTest and CircleCI
- Active community of users on Discord
